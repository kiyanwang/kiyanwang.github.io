# Portfolio Site Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a React portfolio site for kiyanwang.github.io with landing page, projects gallery, and writing feed from Substack RSS.

**Architecture:** Vite + React SPA using HashRouter for GitHub Pages compatibility. Project data lives as YAML files loaded at build time via vite plugin. Substack articles fetched by a pre-build script and saved as JSON. Deployed via `gh-pages` npm package.

**Tech Stack:** Vite, React 18, React Router 6, js-yaml, @originjs/vite-plugin-content (YAML import), gh-pages, vitest + React Testing Library

```js
const projectFiles = import.meta.glob('../data/projects/*.yml', { eager: true });

export function loadProjects() {
  return Object.values(projectFiles).map((mod) => mod.default || mod);
}

export function getFeatured(projects) {
  return projects.filter((p) => p.featured);
}

export function filterByTag(projects, tag) {
  if (tag === 'All') return projects;
  return projects.filter((p) => p.tags.includes(tag));
}

export function getAllTags(projects) {
  const tags = new Set();
  projects.forEach((p) => p.tags.forEach((t) => tags.add(t)));
  return ['All', ...Array.from(tags).sort()];
}
```

### Task 4: Substack RSS fetch script

**Files:**
- Create: `scripts/fetch-articles.mjs`, `src/data/articles.json`

- [ ] **Step 1: Create the fetch script**

Create `scripts/fetch-articles.mjs`:

```js
import { writeFileSync, mkdirSync } from 'fs';
import { join, dirname } from 'path';
import { fileURLToPath } from 'url';

const __dirname = dirname(fileURLToPath(import.meta.url));
const RSS_URL = 'https://virtualchaos.substack.com/feed';
const OUTPUT = join(__dirname, '..', 'src', 'data', 'articles.json');

async function fetchArticles() {
  console.log('Fetching Substack RSS feed...');
  const res = await fetch(RSS_URL);
  if (!res.ok) {
    console.warn(`RSS fetch failed (${res.status}), using empty articles`);
    writeFileSync(OUTPUT, '[]');
    return;
  }
  const xml = await res.text();
  const articles = parseRSS(xml);
  mkdirSync(dirname(OUTPUT), { recursive: true });
  writeFileSync(OUTPUT, JSON.stringify(articles, null, 2));
  console.log(`Wrote ${articles.length} articles to ${OUTPUT}`);
}

function parseRSS(xml) {
  const items = [];
  const itemRegex = /<item>([\s\S]*?)<\/item>/g;
  let match;
  while ((match = itemRegex.exec(xml)) !== null) {
    const block = match[1];
    const title = extractTag(block, 'title');
    const link = extractTag(block, 'link');
    const description = extractTag(block, 'description');
    const pubDate = extractTag(block, 'pubDate');
    if (title && link) {
      items.push({
        title: decodeHTML(title),
        link,
        description: decodeHTML(stripHTML(description || '')).slice(0, 200),
        date: pubDate ? new Date(pubDate).toISOString().split('T')[0] : null,
      });
    }
  }
  return items;
}

function extractTag(xml, tag) {
  const cdataMatch = xml.match(new RegExp(`<${tag}><!\\[CDATA\\[([\\s\\S]*?)\\]\\]></${tag}>`));
  if (cdataMatch) return cdataMatch[1];
  const simpleMatch = xml.match(new RegExp(`<${tag}>([\\s\\S]*?)</${tag}>`));
  return simpleMatch ? simpleMatch[1] : null;
}

function stripHTML(str) {
  return str.replace(/<[^>]*>/g, '').replace(/\s+/g, ' ').trim();
}

function decodeHTML(str) {
  return str
    .replace(/&amp;/g, '&')
    .replace(/&lt;/g, '<')
    .replace(/&gt;/g, '>')
    .replace(/&quot;/g, '"')
    .replace(/&#039;/g, "'")
    .replace(/&#x27;/g, "'");
}

fetchArticles().catch((err) => {
  console.error('Failed to fetch articles:', err.message);
  writeFileSync(OUTPUT, '[]');
});
```
